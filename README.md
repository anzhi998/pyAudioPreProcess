# pyAudioPreProcess
音频信号预处理python代码（做了一些封装)  
## 内有函数：  
1、连续小波变换  
2、得到适用于频谱分析的汉明窗  
3、带通滤波器  
4、对信号滤波  
5、归一化  
6、分帧  
7、对分帧信号加窗  
8、把分帧信号复原   
9、绘制小波时频图  
需要的环境：  
'''  
    python-version>=3.6  
    pip install pywavelets  
    pip install numpy  
    pip install scipy  
    pip install matplotlib  
'''
